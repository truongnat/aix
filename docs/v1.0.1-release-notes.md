# v1.0.1 — Docs, Walkthrough, And Release Polish

**Status:** patch release. No workflow contract change, no provider support expansion, and no new runtime claims.

## What changed

- **README refresh** — clearer open-source positioning, faster quickstart, and a dedicated walkthrough section
- **Landing page polish** — stronger onboarding flow, cleaner open-source product framing, and a more professional section hierarchy
- **Walkthrough video integration** — repo-local MP4 linked from the README and embedded directly on the landing page
- **Release metadata alignment** — version strings, docs index, changelog, and release checklist updated for the next npm publish

## Why this is a patch release

`v1.0.1` improves presentation, documentation clarity, and release readiness without changing the core harness loop:

```text
Session Start → Map → Discuss → Plan → Run → Verify → Ship → Remember
```

The provider support model is unchanged:

- Claude remains the strongest native path
- Cursor, Codex, and Gemini remain adapter or fallback paths
