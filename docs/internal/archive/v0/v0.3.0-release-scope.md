# v0.3.0 Release Scope

## In Scope

- validate this harness repository with `node bin/validate.js`
- validate adopted host repository profile artifacts
- validate adopted host repository goal artifacts
- support `--target`
- support `--profile-only`
- support `--goal`
- structural heading and path checks
- dependency-free tests and fixtures
- documentation for safe target validation usage

## Out Of Scope

- semantic correctness checks
- application code validation
- secret detection engine
- runtime adapter
- server, database, or background process
- large file scanning
- package publishing automation

## Scope Boundary

`v0.3.0` is a lightweight structural validation release.

It is not intended to decide whether an application is correct, secure, production-ready, or well-designed outside the harness artifact contract.
