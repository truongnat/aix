The Pipeline

Four phases. One command. Infinite possibilities.

🧠

Plan

Top-tier model architects the solution with comprehensive technical design, architecture decisions, and success criteria.

📐

Rules

Generates project conventions, tech stack decisions, coding standards, and testing strategy. No questions — only decisions.

🎫

Tasks

Decomposes into an executable kanban of tickets with dependencies, acceptance criteria, and implementation priorities.

🤖

Execute

Coder implements each ticket, runs tests automatically, reviewer scores it. Loops until score ≥ 9/10. You approve at every step.

Why Agentic Goal?

Everything you need for autonomous development

🧠

Multi-Agent Orchestration

Specialized agents for planning, rules, tasks, coding, and review — each with their own models, skills, and capabilities.

💾

Checkpointed Execution

Every step saved to SQLite. Interrupt anytime, review artifacts in your IDE, and resume exactly where you left off.

🔒

Budget & Safety

Hard-stop budget enforcement, sandboxed file operations, command denylists, and configurable limits for safe autonomous execution.

🔄

Human-in-the-Loop

Approve, regenerate with feedback, or skip at every phase. Full control over the autonomous process with numbered menu actions.

🧪

Test-Driven

Coder automatically runs tests before reviewer approval. Test output passed to reviewer for correctness assessment.

🎨

Skills System

Inject domain knowledge via Markdown skill files. YAML frontmatter with tags enables dynamic loading based on ticket context. Import from URL with goal skills import.

⚡

Streaming Output

Watch LLM tokens stream in real-time for plan, rules, and tasks generation. No more staring at spinners.

🛠️

Recovery Commands

Retry failed tickets, skip problematic ones, or abort entire goals. Clean up old workspaces with goal clean.
