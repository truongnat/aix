# 🔥 STEP 16 — Condition Evaluator

Tạo:

```
engine/condition.rs
```

Function:

```rust
pub fn evaluate_condition(
    condition: &str,
    step_results: &HashMap<String, SkillOutput>
) -> Result<bool>
```

Logic:

1. Parse `{{step_id}} == true`
2. Extract step_id
3. Lookup step_results
4. Must be Boolean
5. Compare with literal true/false
6. Return result

---

# 🔥 STEP 17 — Modify Executor

Trong execution loop:

Before executing step:

```rust
if let Some(cond) = &step.condition {
    let should_run = evaluate_condition(cond, &step_results)?;
    if !should_run {
        println!("⏭ Skipping step: {}", step.id);
        continue;
    }
}
```

---

# 🔒 Important Design Rule

Skipped step:

* Không insert result vào step_results
* Nhưng phải mark completed_steps để không block dependent nodes

Nếu không mark completed → dependent node sẽ fail.
