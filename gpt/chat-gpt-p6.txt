# ✅ STEP 6.3 — Modify Skill Trait

Thêm method:

```rust
fn capability(&self) -> &SkillCapability;
```

Giờ trait là:

```rust
pub trait Skill: Send + Sync {
    fn name(&self) -> &str;
    fn capability(&self) -> &SkillCapability;
    fn execute(&self, input: SkillInput, ctx: &mut ExecutionContext) -> Result<SkillOutput>;
}
```

---

# ✅ STEP 6.4 — Introduce Typed IO Wrapper

Tạo:

```rust
pub enum SkillInput {
    Text(String),
    Json(serde_json::Value),
    Number(f64),
    Boolean(bool),
}

pub enum SkillOutput {
    Text(String),
    Json(serde_json::Value),
    Number(f64),
    Boolean(bool),
}
```

Hiện tại workflow vẫn parse input dạng text.
Executor convert Text → SkillInput::Text.
