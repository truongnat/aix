# ✅ 1️⃣ Define Markdown Convention (Stable DSL)

Ta dùng format này:

```markdown
# Workflow: test-workflow
Domain: demo

## Step: step1
Skill: echo
Input:
Hello AI

## Step: step2
Skill: echo
Input:
Second step
```

### Rules:

* `# Workflow:` bắt buộc
* `Domain:` optional
* `## Step:` bắt đầu step
* `Skill:` bắt buộc
* `Input:` dòng kế tiếp là nội dung (cho đến step mới)

Không dùng YAML block.
Không mix frontmatter.

---

# ✅ 2️⃣ Modify Loader (workflow/loader.rs)

Xoá serde_yaml.

Giờ ta:

```rust
pub fn load_from_markdown(path: &str) -> Result<Workflow>
```

Flow:

1. Read file → String
2. Split lines
3. Parse state machine:

State:

```text
Start
InMeta
InStep
InInput
```

Pseudo:

```text
for line:
  if starts_with "# Workflow:"
      parse name
  if starts_with "Domain:"
      parse domain
  if starts_with "## Step:"
      push previous step (if exists)
      create new step
  if starts_with "Skill:"
      set skill
  if starts_with "Input:"
      switch to input mode
  else if in input mode:
      append line to current_step.input
```

Cuối file → push last step.

---

# ✅ 3️⃣ Validation Rules

Trước khi return Workflow:

* workflow.name != empty
* steps.len() > 0
* mỗi step phải có skill

Nếu thiếu → return Err(anyhow!)
