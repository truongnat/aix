# ✅ STEP 8.1 — Topological Sort

Implement Kahn’s algorithm:

```text
queue = nodes with in_degree == 0

while queue not empty:
    pop node
    push to execution order
    reduce in_degree of neighbors
    if in_degree == 0 → push
```

Nếu cuối cùng execution_order.len != total_steps
→ cycle detected → error.

# 🔥 STEP 9 — Modify Executor

Thay:

```rust
for step in workflow.steps
```

Bằng:

```rust
let order = graph.topo_sort()?;

for step_id in order:
    execute(step_id)
```

Step lookup bằng HashMap.

---

# 🔒 Determinism Rule

Nếu nhiều node có in_degree = 0
→ phải sort lexicographically trước khi push queue.

Không để random order.
