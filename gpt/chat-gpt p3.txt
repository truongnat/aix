# ✅ STEP 3.3 — Execution Logic (IMPORTANT)

Pseudo logic:

```
create ExecutionContext
ctx.memory = empty string

for each step:
    print step start
    ctx.step_id = step.id

    skill = registry.get(step.skill)
    output = skill.execute(step.input, &mut ctx)

    append output to ctx.memory
    print output

return ctx.memory
```
