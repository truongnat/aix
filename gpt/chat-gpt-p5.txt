# 🧱 PHASE 2

# Domain-Specific Skill Isolation

🎯 Mục tiêu:

* Mỗi workflow thuộc **một domain**
* Domain chỉ được dùng skill thuộc domain đó
* Runtime phải enforce isolation
* Không cho cross-domain skill injection
* Không để workflow gọi skill ngoài domain

Đây là bước để hệ thống có thể scale về sau.

---

# 🔥 STEP 5 — Redesign Registry Architecture

Hiện tại:

```text
SkillRegistry
  HashMap<String, Arc<dyn Skill>>
```

❌ Vấn đề:

* Global skill pool
* Không domain separation
* Không security boundary

---

# ✅ STEP 5.1 — Introduce `DomainRegistry`

Tạo:

```rust
pub struct DomainRegistry {
    domains: HashMap<String, SkillRegistry>,
}
```

Không dùng nested Arc ở đây.

---

# ✅ STEP 5.2 — Change Workflow Execution Rule

Executor phải:

1. Read `workflow.meta.domain`
2. Lookup domain registry
3. Chỉ dùng skill trong domain đó
4. Nếu domain không tồn tại → Error
5. Nếu skill không thuộc domain → Error

---

# 🧠 Design Decision

Không nên để SkillRegistry global nữa.

Thay vào đó:

```rust
Executor {
    domain_registry: Arc<DomainRegistry>
}
```

---

# ✅ STEP 5.3 — DomainRegistry API

Interface:

```rust
pub fn register_domain(&mut self, name: &str);

pub fn register_skill(
    &mut self,
    domain: &str,
    skill: Arc<dyn Skill>
);

pub fn get_skill(
    &self,
    domain: &str,
    skill: &str
) -> Result<Arc<dyn Skill>>;
```

Rules:

* Domain phải tồn tại trước khi add skill
* Không auto-create domain

---

# ✅ STEP 5.4 — Executor Modification

Trong execute_workflow:

Thay:

```rust
registry.get(step.skill)
```

Bằng:

```rust
domain_registry.get_skill(workflow.meta.domain, step.skill)
```

---

# ✅ STEP 5.5 — CLI Wiring Change

Trong main:

Thay vì:

```rust
let mut registry = SkillRegistry::new();
registry.register(EchoSkill);
```

Giờ phải:

```rust
let mut domains = DomainRegistry::new();
domains.register_domain("demo");
domains.register_skill("demo", Arc::new(EchoSkill));
```

Workflow markdown phải có:

```text
Domain: demo
```

Nếu không có domain → Error.

* Workflow domain = finance
* Không thể gọi skill thuộc domain = healthcare
* Không thể gọi skill global

Runtime trở thành domain-aware execution engine.
