# 🔥 STEP 25 — Step Execution Wrapper

Tách logic:

```rust
async fn execute_with_retry(
    step: &WorkflowStep,
    ...
) -> Result<SkillOutput>
```

Logic:

```
attempt = 0
loop:
    result = skill.execute(...)
    if Ok → return
    if attempt < retry:
        attempt += 1
        continue
    else:
        return Err
```

# 🔥 STEP 26 — Failure Handling in Batch

Trong parallel batch:

Sau khi `join_all`:

For each result:

### Case 1 — Success

* store output
* mark completed

### Case 2 — Failure

If strategy == FailFast:

* Immediately return Err
* Abort entire workflow

If strategy == Continue:

* Print error
* Mark completed
* Do NOT insert result
* Downstream node:

  * If reference to this step → error
  * Otherwise may still run

---

# 🔒 Propagation Rule

If a step failed with Continue:

* It is considered completed
* But no output exists
* Reference to it → error
* Condition referencing it → error
