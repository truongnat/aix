# Dev Video Idea Lab Prompt

Use this template to generate a repeatable video concept backlog from real project evidence.

Task context should include:
- target audience (engineers, CTO, founders, security team, etc.)
- primary conversion goal (demo request, repo star, signup, contact)
- preferred channels (YouTube, LinkedIn, X, short-form)
- publishing cadence and team capacity
- key proof artifacts to emphasize (CLI commands, trace output, landing page, workflow files)

Output expectations:
- 10-15 concepts with clear hook and narrative
- every concept must include all 4 required beats:
  1) structure
  2) CLI
  3) use case
  4) success output (landing page)
- top 3 concepts expanded into production-ready Remotion plan
- 2-week prioritized backlog with success metrics
