# Dev Build Landing Page Prompt

Use this template when building a landing page feature end-to-end.

Task context should include:
- product/service name
- target audience
- core conversion goal (signup/demo/contact/purchase)
- existing stack or file constraints
- visual direction (if any)
- non-functional constraints (SEO/performance/accessibility)

Expected output:
- architecture/spec for landing page sections
- implementation-ready file plan
- validation/test checklist
- final execution handoff
