# Dev Project Intelligence Prompt

Use this template to onboard and upgrade an existing repository.

Task should include:
- what the project is trying to achieve
- current pain points (speed, bugs, delivery, architecture, testing, ops)
- desired outcomes in 2-4 weeks
- constraints (time, budget, team size, stack limitations)

Expected output:
- fast project understanding
- best-fit workflows to run next
- best-fit skills to adopt
- upgrade roadmap for solo developer execution
