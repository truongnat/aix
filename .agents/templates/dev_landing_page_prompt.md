# Dev Landing Page Prompt

Use this template when the task is to build or redesign a landing page feature.

Required context to include in task:
- product/service name
- target audience
- primary conversion goal (signup/demo/contact/purchase)
- visual direction (if any)
- constraints (framework, deadline, performance, SEO, accessibility)

Expected output quality:
- architecture spec
- file-level implementation handoff
- validation/test checklist
- clear done criteria
