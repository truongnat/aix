# AI Design Tool Comparison

## Image Generation

| Tool | API | Cost | Best for | Limitations |
|------|-----|------|----------|-------------|
| **Midjourney v6** | Unofficial only | $10–$60/mo | Highest aesthetic quality | Discord-only, no official API |
| **DALL-E 3** | OpenAI API | $0.04–$0.12/img | Brand prompting, text in images | Moderate aesthetics vs MJ |
| **Stable Diffusion** | Local / Replicate | Free (local) | Custom fine-tuning, ControlNet | Setup complexity |
| **Adobe Firefly** | Adobe API | CC subscription | Commercial safety, PS integration | Requires Creative Cloud |
| **Ideogram** | API available | $7–$16/mo | Text rendering in images | Less style control |

## Video / Motion Generation

| Tool | Input | Max duration | Best for |
|------|-------|-------------|----------|
| **Runway Gen-2** | Text / Image | 16s | High quality motion |
| **Pika Labs** | Text / Image | 3–10s | Fast iteration, affordable |
| **Stable Video Diffusion** | Image | 4s | Local, free, consistent |
| **Sora** (OpenAI) | Text | 60s | Long-form (limited access) |
| **Kling** | Text / Image | 30s | Realistic motion |

## AI Prototyping / Wireframing

| Tool | Output | Best for |
|------|--------|----------|
| **Galileo AI** | Figma design | Full UI from description |
| **Uizard** | Figma / HTML | Sketch-to-wireframe, team collab |
| **v0.dev** (Vercel) | React/Tailwind code | Code-first UI generation |
| **Figma AI** | Figma layers | In-Figma generation, plugin |
| **Framer AI** | Framer site | Marketing pages |

## Color & Typography

| Tool | Best for |
|------|----------|
| **Khroma** | AI palette from brand preferences |
| **Adobe Firefly Color** | Brand-safe, integrated with Adobe |
| **Coolors AI** | Fast palette iteration |
| **Fontjoy** | AI font pairing |
| **Scripts/generate-color-palette.js** | Programmatic palette generation |

## Selection guide

```
Need commercial-safe assets?   → Adobe Firefly
Need highest quality images?   → Midjourney v6
Need API automation?           → DALL-E 3 or Stability AI API
Need free + customizable?      → Stable Diffusion (local)
Need video clips?              → Runway Gen-2
Need UI code from description? → v0.dev
Need Figma output?             → Galileo AI or Figma AI
```
