# Skill Packs

Skill packs are compact routing aids that help an agent choose the most relevant core skills and commands for a task domain.

## Included Packs

- [Frontend](frontend.md)
- [Backend](backend.md)
- [Mobile](mobile.md)
- [DevOps](devops.md)
- [Debugging](debugging.md)
- [Data & AI](data-ai.md)
- [Security](security.md)
- [Cloud](cloud.md)

## Pack Rules

- Packs do not replace the core skills.
- Packs should stay small and practical.
- Packs should route agents toward relevant commands, checks, and verification expectations.
- Deep domain-specific skills can be added later only when there is a clear recurring need.
