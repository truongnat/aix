# Public assets

Large media (walkthrough MP4) is **not** stored in git. It is published as a [GitHub Release asset](https://github.com/truongnat/ai-engineering-harness/releases).

Canonical URL: see `media/walkthrough-video.json`.

Maintainers upload with:

```bash
node scripts/publish-walkthrough-video.js --tag v1.0.1 --file /path/to/AI_Engineering_Harness.mp4
```
